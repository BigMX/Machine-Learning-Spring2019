Pre-Processing

In this part we will define our categorical headers and numeric headers, we will first transform all categorical headers into integers so that our keras model will recognize them as one hotted encoded categorical columns. We will also use the standard scalar to preprocess the numeric data to make all the data centered around 0 and have variance in the same order.

According to https://scikit-learn.org/stable/modules/generated/sklearn.preprocessing.StandardScaler.html, "If a feature has a variance that is orders of magnitude larger that others, it might dominate the objective function and make the estimator unable to learn from other features correctly as expected".Here in our dataset, a few numeric data contain the exact "dominating" values. Therefore, we use the standard scalar to elimniate them.

Evaluation Metrics

In this lab, we will use recall as our evaluation metric, the recall score is calculated by TP/(TP+FN). In our business case, accurate is not a good metric because we have an imbalance dataset. That is there are more people that choose not to subscribe the bank's deposit than those who will subscribe. Here, banks focus on if a client will subscribe a deposit and will do analysis and plan for them in ahead. Therefore, a recall is good metric because it can indicate the ratio of false negative because banks certainly do not want to waste more time on people that will not subscribe the deposit, but it is okay for them to miss a few clients because banks can still process their subscription after they actually subscribe the deposit.

from keras import backend as K

def recall_m(y_true, y_pred):
        true_positives = K.sum(K.round(K.clip(y_true * y_pred, 0, 1)))
        possible_positives = K.sum(K.round(K.clip(y_true, 0, 1)))
        recall = true_positives / (possible_positives + K.epsilon())
        return recall

cross_columns = [['education','job'],

embed_branches = []
X_ints_train_1 = []
X_ints_validation_1 = []
X_ints_test_1=[]
all_inputs_1 = []
all_wide_branch_outputs_1 = []

for cols in cross_columns:
    # encode crossed columns as ints for the embedding
    enc = LabelEncoder()
    
    # create crossed labels
    X_crossed_train = df_train[cols].apply(lambda x: '_'.join(x), axis=1)
    X_crossed_validation = df_validation[cols].apply(lambda x: '_'.join(x), axis=1)
    X_crossed_test = df_test[cols].apply(lambda x: '_'.join(x), axis=1)
    
    enc.fit(np.hstack((X_crossed_train.values,  X_crossed_validation.values)))
    X_crossed_train = enc.transform(X_crossed_train)
    X_crossed_validation = enc.transform(X_crossed_validation)
    X_crossed_test = enc.transform(X_crossed_test)
    X_ints_train_1.append( X_crossed_train )
    X_ints_validation_1.append( X_crossed_validation )
    X_ints_test_1.append(X_crossed_test)
    
    # get the number of categories
    N = max(X_ints_train_1[-1]+1) # same as the max(df_train[col])
    
    # create embedding branch from the number of categories
    inputs = Input(shape=(1,),dtype='int32', name = '_'.join(cols))
    all_inputs_1.append(inputs)
    x = Embedding(input_dim=N, 
                  output_dim=int(np.sqrt(N)), 
                  input_length=1, name = '_'.join(cols)+'_embed')(inputs)
    x = Flatten()(x)
    all_wide_branch_outputs_1.append(x)
    
# merge the branches together
wide_branch = concatenate(all_wide_branch_outputs_1, name='wide_concat')
wide_branch = Dense(units=1,activation='sigmoid',name='wide_combined')(wide_branch)

# reset this input branch
all_deep_branch_outputs = []
# add in the embeddings
for col in categorical_headers_ints:
    # encode as ints for the embedding

    X_ints_validation_1.append( df_validation[col].values )

# also get a dense branch of the numeric features
