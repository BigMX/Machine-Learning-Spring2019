Preparation and Overview

The quality of red wine is determined by many factors. Because each red wine has components in different concentration, the red wine is likely to taste differently from brand to brand. A red wine connoisseur can usually tell the age, the quality, maybe even the name of the wine. People who know how to examine the quality of a red wine are usually considered elegant and noble. Sometimes, if people know nothing about wine, they will probably purchase red wine in bad quality for their grand event. If people just want to throw a small party, it is unwise to buy the best red wine. To solve this problem, in this lab, we will use a machine learning training model to become the wine connoisseur. We will use a dataset of red wine. This dataset is related to red variants of the Portuguese "Vinho Verde" wine. In our training process, we will classify the red wines into three quality classes: "Good", "Medium", and "Bad".

Like mentioned earlier, any party host will be interested in our results. Using our results, he/she will have freedom to choose the quality of wine for the upcoming party. Also, any wine taster will be interested in our product because it can be used to train his/her tastebuds. Most importantly, red wine manufacturers will be interested so that they can know what quality they have and know how to improve the quality by adjusting the recipe.

Our training model will mainly be deployed in real-time because the whole dataset is the same type of wine and our model is just for this specific brand. The quality or grade varies by time and storing condition.

Measure of Success

Our prediction task will be to "grade" the different red wine. Despite wine experts lauding their tastebuds and elevated senses, a study conducted by Robert Hodgson proved otherwise. He approached the organizers of the California state fair wine competition and asked them to present judges with different samples from the same bottle three times. Normally, judges scale a wine from range 50-100. He claims the results were "disturbing". After giving the judges samples from the same wine bottle, he found that they each scored it differently each time! Ranging from low 80s to high 90s, the expert judges (winemakers, sommeliers, critics and wine consultants and academics) were highly inconsistent. He further went on to say that objectively evaluating large numbers of wines as currently attempted at wine competitions is, "beyond human ability". Due to this fact that not even "experts" can consistently and accurately identify and grade wines, our goal is to reach 70% accuracy.

Data Understanding

  • fixed acidity(float): the fixed acidity of most acids involved with wine
  • volatile(float): the amount of acetic acid in wine, which at too high of levels can lead to an unpleasant, vinegar taste
  • citric acid(float): found in small quantities, citric acid can add 'freshness' and flavor to wines
  • residual sugar(float): the amount of sugar remaining after fermentation stops
  • chlorides(float): the amount of salt in the wine
  • free sulfur dioxide(float): the free form of SO2 exists in equilibrium between molecular SO2 (as a dissolved gas) and bisulfite ion
  • total sulfur dioxide(float): amount of free and bound forms of S02
  • density(float): the density of water is close to that of water depending on the percent alcohol and sugar content
  • pH(float): describes how acidic or basic a wine is on a scale from 0 (very acidic) to 14 (very basic)
  • sulphates(float): a wine additive which can contribute to sulfur dioxide gas (S02) level
  • alcohol(float): the percent alcohol content of the wine
  • quality(int,ordinal): The actual quality of the wine, ranging from 0-10 (This will be dropped)
  • class(int,ordinal): Our prediction, there are three classes: 0, 1, 2. (This is our prediction task)

Metrics

For our metrics, we will implement a customized evaluation metric. People use our model to predict the classification of red wine. We have three classes of quality 0, 1, 2. Classifying 0 as 1 or 1 as 2 are not considered serious mistakes. However, classifying 0 as 2, or 2 as 0 are considered serious misclassifications. Therefore, we decide to use a weighted accuracy score to evaluate our performance. We will add weight to each classification situation so that a grievous mistake will affect the score greatly, while providing significant leeway for minor mistakes. Additionally, our dataset has a large imbalance in distribution. Therefore, it is a good idea to give more weight to the minority class. In our customized metrics, we give more weight to the minority class and significant misclassifications.

def weighted_accuracy_score(y,yhat):
    score=0
    for i in range(len(y)):
        if y[i] == 0:
            if yhat[i] == 0:
                score+=10
            elif yhat[i] == 1:
                score-=0
            elif yhat[i] == 2:
                score-=4
        elif y[i] == 1:
            if yhat[i] == 0:
                score-=0
            elif yhat[i] == 1:
                score+=1
            elif yhat[i] == 2:
                score-=0
        elif y[i] == 2:
            if yhat[i] == 0:
                score-=4
            elif yhat[i] == 1:
                score-=0
            elif yhat[i] == 2:
                score+=1
    return score/len(y)

class TwoLayerPerceptronBase(object):
    def __init__(self, n_hidden=30,
                 C=0.0, epochs=500, eta=0.001, random_state=None):
        np.random.seed(random_state)
        self.n_hidden = n_hidden
        self.l2_C = C
        self.epochs = epochs
        self.eta = eta
        
    @staticmethod
    def _encode_labels(y):
        """Encode labels into one-hot representation"""
        onehot = pd.get_dummies(y).values.T
            
        return onehot

    def _initialize_weights(self):
        """Initialize weights with small random numbers."""
        W1_num_elems = (self.n_features_ + 1)*self.n_hidden
        W1 = np.random.uniform(-1.0, 1.0, size=W1_num_elems)
        W1 = W1.reshape(self.n_hidden, self.n_features_ + 1) # reshape to be W
        
        W2_num_elems = (self.n_hidden + 1)*self.n_output_
        W2 = np.random.uniform(-1.0, 1.0, size=W2_num_elems)
        W2 = W2.reshape(self.n_output_, self.n_hidden + 1)
        return W1, W2
    
    @staticmethod
    def _sigmoid(z):
        """Use scipy.special.expit to avoid overflow"""
        # 1.0 / (1.0 + np.exp(-z))
        return expit(z)
    
    @staticmethod
    def _add_bias_unit(X, how='column'):
        """Add bias unit (column or row of 1s) to array at index 0"""
        if how == 'column':
            ones = np.ones((X.shape[0], 1))
            X_new = np.hstack((ones, X))
        elif how == 'row':
            ones = np.ones((1, X.shape[1]))
            X_new = np.vstack((ones, X))
        return X_new
    
    @staticmethod
    def _L2_reg(lambda_, W1, W2):
        """Compute L2-regularization cost"""
        # only compute for non-bias terms
        return (lambda_/2.0) * np.sqrt(np.mean(W1[:, 1:] ** 2) + np.mean(W2[:, 1:] ** 2))
    
    def _cost(self,A3,Y_enc,W1,W2):
        '''Get the objective function value'''
        cost = np.mean((Y_enc-A3)**2)
        L2_term = self._L2_reg(self.l2_C, W1, W2)
        return cost + L2_term

class TwoLayerPerceptron(TwoLayerPerceptronBase):
    def _feedforward(self, X, W1, W2):
        """Compute feedforward step
        -----------
        X : Input layer with original features.
        W1: Weight matrix for input layer -> hidden layer.
        W2: Weight matrix for hidden layer -> output layer.
        ----------
        a1-a3 : activations into layer (or output layer)
        z1-z2 : layer inputs 

        """
        A1 = self._add_bias_unit(X.T, how='row')
        Z1 = W1 @ A1
        A2 = self._sigmoid(Z1)
        A2 = self._add_bias_unit(A2, how='row')
        Z2 = W2 @ A2
        A3 = self._sigmoid(Z2)
        return A1, Z1, A2, Z2, A3
    
    def _get_gradient(self, A1, A2, A3, Z1, Z2, Y_enc, W1, W2):
        """ Compute gradient step using backpropagation.
        """
        # backpropagation
        grad1 = np.zeros(W1.shape)
        grad2 = np.zeros(W2.shape)
        
        # for each instance's activations 
        for (a1,a2,a3,y) in zip(A1.T,A2.T,A3.T,Y_enc.T):
            dJ_dz2 = -2*(y - a3)*a3*(1-a3)
            dJ_dz1 = dJ_dz2 @ W2 @ np.diag(a2*(1-a2))
                         
            grad2 += dJ_dz2[:,np.newaxis]  @ a2[np.newaxis,:]
            grad1 += dJ_dz1[1:,np.newaxis] @ a1[np.newaxis,:] 
            # don't incorporate bias term in the z1 gradient 
            # (its added in a2 from another layer)
            

        # regularize weights that are not bias terms
        grad1[:, 1:] += (W1[:, 1:] * self.l2_C)
        grad2[:, 1:] += (W2[:, 1:] * self.l2_C)

        return grad1, grad2
    
    def predict(self, X):
        """Predict class labels"""
        _, _, _, _, A3 = self._feedforward(X, self.W1, self.W2)
        y_pred = np.argmax(A3, axis=0)
        return y_pred
    
    
    def fit(self, X, y, print_progress=False):
        """ Learn weights from training data."""
        
        X_data, y_data = X.copy(), y.copy()
        Y_enc = self._encode_labels(y)
        
        # init weights and setup matrices
        self.n_features_ = X_data.shape[1]
        self.n_output_ = Y_enc.shape[0]
        self.W1, self.W2 = self._initialize_weights()

        self.cost_ = []
        for i in range(self.epochs):

            if print_progress>0 and (i+1)%print_progress==0:
                sys.stderr.write('\rEpoch: %d/%d' % (i+1, self.epochs))
                sys.stderr.flush()

            # feedforward all instances
            A1, Z1, A2, Z2, A3 = self._feedforward(X_data,self.W1,self.W2)
            
            cost = self._cost(A3,Y_enc,self.W1,self.W2)
            self.cost_.append(cost)

            # compute gradient via backpropagation
            grad1, grad2 = self._get_gradient(A1=A1, A2=A2, A3=A3, Z1=Z1, Z2=Z2, Y_enc=Y_enc,
                                              W1=self.W1, W2=self.W2)

            self.W1 -= self.eta * grad1
            self.W2 -= self.eta * grad2
            

        return self

class AdvancedMLP(TwoLayerPerceptronVectorized):
    def __init__(self, act='sigmoid',cost='quadratic',layer=2,**kwds):        
        # need to add to the original initializer 
        self.layer=layer+1  #Total Layer= Hidden layer+1
        self.act=act
        self.cost=cost
        self.W=[]
        self.gradient_epochs=[]
        # but keep other keywords
        super().__init__(**kwds)
        
    def _activation(self, z):
        if self.act == 'sigmoid':
            return self._sigmoid(z)
        else:
            return z/np.max(z)
        
    def _feedforward(self, X):
        A=[]
        Z=[]
        a = self._add_bias_unit(X.T, how='row')
        A.append(a)
        #A has one more element than Z, the last element is output
        for i in range(self.layer-1):
            z = self.W[i] @ a
            input_z=self._activation(z)
            a = self._add_bias_unit(input_z, how='row')
            A.append(a)
            Z.append(z)
        z = self.W[self.layer - 1] @ a #last layer
        a = self._sigmoid(z)                  #output
        A.append(a)
        Z.append(z)
        return A,Z

    def fit(self,X,y,print_progress=0):

        Z=[]
        X_data, y_data = X.copy(), y.copy()
        Y_enc = self._encode_labels(y)

        self.n_output_ = Y_enc.shape[0]

class More_AdvancedMLP(AdvancedMLP):
    
    def _activation(self, z):
        if self.act == 'sigmoid':
            return self._sigmoid(z)
        elif self.act == 'linear':
            return z/np.max(z)
        elif self.act == 'relu':
            return z*(z>0)/np.max(z)
        elif self.act == 'silu':
            return z*self._sigmoid(z)/np.max(z)
        else:
            return z
        
    def bp_activation(self,A):
        if self.act=='sigmoid':
            return A*(1-A)
        elif self.act=='linear':
            return np.ones(A.shape)
        elif self.act=='relu':
            return np.ones(A.shape)
        elif self.act == 'silu':
            return A*self._sigmoid(A)+self._sigmoid(1-A*self._sigmoid(A))
        else:
            return A
